========================================
  Weiquanbao - Chat Record Export Tool
========================================

FEATURES:
  - Export chat records from DingTalk, WeChat, QQ, Feishu
  - Smart keyword search
  - AI legal analysis
  - Batch export to JSON

HOW TO USE:

  Method 1: Double-click start.bat (Recommended)
  -----------------------------------------------
  1. Double-click "start.bat"
  2. Wait for dependencies to install (first run: 2-3 min)
  3. Open browser: http://localhost:8090

  Method 2: Build EXE (No Python needed)
  -----------------------------------------------
  1. Double-click "build.bat"
  2. Wait 3-5 minutes
  3. Find "weiquanbao.exe" in dist folder
  4. Double-click exe to run

PLATFORM GUIDE:

  DingTalk: Auto-detected, works immediately
  WeChat: Must be running, auto-extracts keys
  QQ: Export TXT from QQ -> Import in Weiquanbao
  Feishu: Requires Feishu Open Platform app setup

FAQ:

  Q: WeChat says "key not found"?
  A: Make sure WeChat is running and logged in

  Q: How to export QQ chat?
  A: QQ -> Message Manager (Ctrl+Shift+Z) -> Right-click -> Export -> TXT

  Q: Where are exported files?
  A: In the "data/exports" folder
