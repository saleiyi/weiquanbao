🛡️ 维权宝

一站式导出钉钉、微信、QQ、飞书聊天记录，AI 智能分析维权要素，生成专业维权方案

📥 下载使用 ⭐ GitHub

🚀 核心功能

从数据提取到维权方案，全流程覆盖

📱

多平台支持

钉钉、微信、QQ、飞书四大平台聊天记录一站式管理，跨平台搜索

🤖

AI 智能分析

接入 OpenAI / DeepSeek / 智谱等大模型，自动识别案件类型、提取关键事实、评估证据强度

📊

证据评估

自动分析证据强度，识别证据缺口，给出补充建议，帮你构建完整证据链

⚖️

法律依据

根据案件类型自动匹配相关法律条文，引用具体法条，让维权有据可依

📋

时间线生成

从聊天记录中自动提取关键时间节点，按重要性排序，还原事件全貌

💰

赔偿计算

根据案情智能计算赔偿/补偿预期，N+1、退一赔三、逾期利息等自动识别

📱 平台支持

四大平台，各有特色

平台使用方式状态
钉钉 自动检测本地数据，开箱即用 ✅ 最简单
微信 自动从内存提取密钥，需管理员权限 ✅ 简单
飞书 通过开放平台 API 访问，一键配置 ⚠️ 需配置
QQ 借助 qq-chat-exporter 先导出再导入 ⚠️ 需先导出
